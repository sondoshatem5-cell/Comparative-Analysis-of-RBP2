# Comparative-Analysis-of-RBP2
## Overview
This repository contains a team project completed as part of the Principles of Bioinformatics (BINF 301) course at Misr University for Science and Technology (MUST).

## Project Type
Academic Team Project

## My Contribution
My primary contribution was performing the BLAST sequence analysis, including:

- Running BLASTP using the human RBP2 protein sequence.
- Comparing RBP2 across multiple vertebrate species.
- Evaluating sequence similarity using identity, positives, E-value, and query coverage.
- Interpreting BLAST results to support comparative sequence analysis.
